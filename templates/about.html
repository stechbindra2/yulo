About the Application

PCB Defect Detection System

An automated system for detecting manufacturing defects in printed circuit boards.

How It Works

This application uses YOLOv8, a state-of-the-art object detection model, to identify and locate common defects in PCB images. The model has been trained on a dataset of PCB images with various defects to accurately classify and detect issues.

Detectable Defects

Missing Hole

Absence of a required hole in the PCB.

Mouse Bite

Edge of the PCB is corroded or improperly formed.

Open Circuit

Break in the conductive pathway.

Short

Unintended connection between two points.

Spur

Unwanted extension from a conductive path.

Spurious Copper

Unwanted copper in non-conductive areas.

Model Details

The detection model uses YOLOv8, a computer vision model architecture known for its accuracy and speed in object detection tasks. The model has been fine-tuned specifically for PCB defect detection to achieve optimal performance.

Usage Instructions

  1. Upload a PCB image by clicking "Browse Files" or drag and drop
  2. Click "Detect Defects" to analyze the image
  3. View the detection results with highlighted defects
  4. Save or download the results if needed
